About Me

I am an Azure Data Engineer with over 2.8 years of hands-on experience in designing, developing, and optimizing data solutions on the Microsoft Azure platform. At Capgemini, I have played a key role in two significant projects. The first as a Azure Data Engineer involving data migration from legacy on-premises systems to Azure SQL Database, where I utilized Azure Data Factory and Databricks to build efficient data pipelines focused on data cleansing, transformation, and analysis, while maintaining a seamless data flow. The second project centers on advanced analytics using Splunk for high-profile clients such as Toll, Bayer, and CSL Behring. In this role, I leverage Search Processing Language (SPL) to analyze unstructured data and implement critical features like anomaly detection, dynamic thresholding, and predictive modeling through Splunk's Machine Learning Toolkit (MLTK) to reduce incident overhead and improve operational efficiency. I am proficient in Azure Data Factory, Azure Data Lake Storage, SQL, Python, SPL, Databricks, and MLTK, which enables me to deliver comprehensive data engineering and analytics solutions that meet complex business needs.
